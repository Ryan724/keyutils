keyutils
========

针对键盘事件监听处理的工具库

 keyUtils 是用js对键盘事件的监听
 使用方法：

*  给按键绑定特定函数，例如：
```javascript
  <script src="./libs/index.js"></script>
  
  keyutils.bind("m",function(){
  	console.log("hello world")
  })
```
